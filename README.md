# 🔐 React Password Manager

A modern Password Generator built with **React**, **Vite**, and **Tailwind CSS**. Generate secure passwords with customizable length, numbers, and special characters, and copy them to the clipboard with a single click.

## 🚀 Features

* Generate strong random passwords
* Adjustable password length
* Include/exclude numbers
* Include/exclude special characters
* One-click copy to clipboard
* Responsive UI
* Built with React Hooks (`useState`, `useEffect`, `useCallback`, `useRef`)

## 🛠️ Tech Stack

* React
* Vite
* Tailwind CSS
* JavaScript (ES6+)


## 🔧 How It Works

1. Select the desired password length.
2. Enable numbers if needed.
3. Enable special characters if needed.
4. Password is generated automatically.
5. Click **Copy** to copy the password to your clipboard.

## 🎯 Learning Concepts

This project demonstrates:

* React State Management (`useState`)
* Memoization using `useCallback`
* Side Effects using `useEffect`
* DOM References using `useRef`
* Clipboard API
* Tailwind CSS Styling
* Component-Based UI Development
